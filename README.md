# Smart Parking System

## Overview
The Smart Parking System is a Python-based application designed to optimize parking space management in urban areas. It provides real-time information about available parking spots, helping users find parking easily and efficiently.

## Features
- **Real-time Parking Availability**: Displays current availability of parking spaces.
- **User-Friendly Interface**: Simple and intuitive interface for users to navigate.
- **Data Management**: Utilizes JSON for data storage and retrieval.

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/KARNATI-SREERAM/SE_Group_6.git
   ```
2. Navigate to the project directory:
   ```bash
   cd SE_Group_6
   ```

## Usage
To run the application, execute the following command:
```bash
python smart_parking.py
```

## Data
The application uses a JSON file located in the `data/` directory for storing parking data. Ensure that the `data.json` file is properly formatted and contains the necessary information.
